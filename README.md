# Wise Platform Operations Dashboard

**A demonstration of operational thinking for the Business Operations Manager role at Wise**

What This Is

This is an interactive dashboard that demonstrates how I would approach operational visibility and partner experience management for Wise Platform. Built as part of my application to show concrete thinking rather than just talking about it.

 View Live Demo


 What It Demonstrates

- How I'd create visibility between Product-Engineering and Operations teams
- My framework for identifying and prioritizing operational bottlenecks
- Approach to demand-capacity planning for scaling partner operations
- Understanding of enterprise partner needs (SLAs, communication, reliability)

 Built With

- Vanilla JavaScript
- Chart.js for visualizations
- HTML5/CSS3
- Deployed on Netlify
